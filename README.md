
# Personalized Learning Recommendation System

## Overview

This project implements a Personalized Learning Recommendation System using Machine Learning techniques. The system recommends relevant online courses based on user skills and course similarity.

## Dataset

The project uses a Coursera course dataset containing course information such as:

* Course Name
* Skills
* Rating
* Difficulty Level
* Duration
* Certificate Type
* Partner Organization

A subset of 750 courses was used for model development.

1. Data Collection and Preprocessing

## Features

* Skill-Based Recommendation
* Visualization of Course Data
